## 说明

- ✅ **Canvas**: 完全可用，支持 text/file/link/group 节点、边连接、6 种颜色预设
- ✅ **Graphviz**: 完全可用，支持 DOT 语言、多种布局引擎（dot, neato, fdp 等）
- ✅ **Infographic**: 完全可用，支持 6 大类模板（列表、序列、对比、时间线、步骤、网格）
- ✅ **Vega/Vega-Lite**: 完全可用，支持柱状图、折线图、散点图等所有图表类型

## 使用提示

1. **重新加载插件**: 按 `Ctrl + R`（Windows）/ `Cmd + R`（Mac）
2. **切换到阅读模式**: 按 `Ctrl + E` 或点击右上角眼睛图标
3. **查看插件设置**: 设置 → 第三方插件 → Markdown Visualizer → 齿轮图标
4. **查看控制台**: `Ctrl + Shift + I` → Console 选项卡（如果有错误）
